By using TabGarb you can easily convert your simple post/page contents into javaScript tab contents . 

== Description ==

TabGarb is a WordPress plugin which gives your wordpress a power to change content into Javascript tab enable content very easily, Tabgarb use HTML comment tags in place of using tags like [ (tabs), [tabs] , {tabs} ] which make your content wordpress friendly ( TabGarb Formatting tag will not spoil your content in case TabGarb is not installed in your wordpress ).
TabGarb come with editable css theme ( you can edit tabs css from your admin panel easly ) and you can change the speed and effect of changing tab direct from TabGarb Settings in WordPress admin.

More information will be found [here](http://webgarb.com/freebies/wordpress-plugin/tabgarb/)

== Installation ==

This section describes how to install the plugin and get it working.

1. Download TabGarb and unzip it .
2. Upload `tabgarb` folder to the `/wp-content/plugins/` directory in your server.
3. Activate the TabGarb plugin under the ‘Plugins’ menu in WordPress.
